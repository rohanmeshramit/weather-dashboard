# Weather Dashboard

A weather dashboard web application built with vanilla HTML, CSS and JavaScript. It fetches real-time weather data from the OpenWeatherMap API and displays current conditions, a 5-day forecast, and an hourly temperature graph.

## Live Demo
[Weather Dashboard Live Demo](https://rohanmeshramit.github.io/weather-dashboard/)

## Demo
![Weather Dashboard Demo](assets/demo.gif)

## Features

- Search weather for any city worldwide
- Auto-detects user location on page load
- Displays temperature, feels like, humidity, wind speed, visibility and condition
- 5-day weather forecast cards
- Interactive hourly temperature graph using Chart.js
- Last 5 searched cities saved and clickable
- Severe weather alert banner for extreme conditions
- Auto dark/light mode based on day/night at searched location
- Fully responsive design for mobile and desktop

## Tech Stack

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
- Chart.js

## Setup Instructions

1. Clone the repository
2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Open `script.js` and replace the API key on line 1
```javascript
   const API_KEY = "your_api_key_here";
```
4. Open `index.html` with Live Server or any browser
